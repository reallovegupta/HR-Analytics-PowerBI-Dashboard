**Author:** Abhishek Gupta

## Project Overview

This project presents an **HR Analytics Dashboard built using Microsoft Power BI**.
The dashboard helps organizations analyze employee data to understand workforce trends such as **employee attrition, department distribution, and salary insights**.

The goal of this project is to support **data-driven HR decision making** through interactive visualizations.
